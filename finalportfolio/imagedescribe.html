Get and Set Up the Image Describe Shortcut

Download the Shortcut

Use the following link to download the Image Describe shortcut: Get Image Describe Shortcut (Opens in a new tab)

  1. Once the page loads, click the "Get Shortcut" link
  2. On the next screen, click "Add Shortcut
  3. Continue with the setup steps below

Set Up the Shortcut

Follow these steps to get the Image Describe shortcut up and running with VoiceOver.

  1. Open the Settings app.
  2. Select Accessibility.
  3. Select VoiceOver.
  4. Scroll down the page until you find the option labeled "Commands".
  5. Once in the Commands screen, select the option labeled "All Commands".
  6. Select the option that says "Shortcuts".
  7. Find the "Image Describe" shortcut from the list of available shortcuts and select it.
  8. On this next screen, you can choose to add a touch gesture or keyboard gesture. Make your selection.
  9. On this next screen, pick your preferred gesture.
  10. You're all done. Next time you are on an element or image where you want a description, perform your gesture. After a brief wait, move your focus near the top of the screen where you should find an image description and a text box where you can reply to the AI and ask more questions.
